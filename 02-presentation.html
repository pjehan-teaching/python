Introduction

Il a été créé par Guido van Rossum en 1991 et est aujourd'hui l'un des langages les plus populaires au monde.

Python est un langage de haut niveau, ce qui signifie qu'il est plus proche du langage humain que du langage machine.

Python est un langage multi-paradigme, ce qui signifie qu'il supporte plusieurs styles de programmation (impératif, orienté objet, fonctionnel...).

Python est un langage de programmation interprété, c'est-à-dire qu'il est exécuté ligne par ligne par un interpréteur.

Installation

Windows :

  • Installer Scoop : https://scoop.sh/
  • Installer Python : scoop install python

MacOS :

  • Installer Homebrew : https://brew.sh/
  • Installer Python : brew install python

Linux :

  • Installer Python : sudo apt install python3

Pour vérifier que Python est bien installé, vous pouvez utiliser la commande suivante :

python --version

IDE

Il existe de nombreux IDE (Environnement de Développement Intégré) pour Python, parmi les plus populaires :

  • Visual Studio Code : un éditeur de code léger et extensible, avec de nombreuses extensions pour Python.
  • Jupyter Notebook : un environnement interactif pour le développement de code Python, idéal pour l'analyse de données et le machine learning.
  • PyCharm : un IDE complet pour Python, avec de nombreuses fonctionnalités avancées.
  • ...

Visual Studio Code étant l'IDE le plus utilisé, nous allons l'utiliser pour ce cours.

Pour installer l'extension Python pour Visual Studio Code, vous pouvez suivre les étapes suivantes :

  1. Ouvrir Visual Studio Code
  2. Cliquer sur l'icône Extensions dans la barre latérale gauche
  3. Rechercher "Python" dans la barre de recherche
  4. Installer l'extension Python officielle de Microsoft
