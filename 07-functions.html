Fonctions

En Python, les fonctions sont définies à l'aide du mot-clé def : 

def greet(name: str) -> None:
    print("Hello, " + name)

greet("World")

Fonctions

Une fonction peut retourner une valeur à l'aide du mot-clé return : 

def add(a: int, b: int) -> int:
    return a + b

result: int = add(5, 3)
print(result)

Récursivité

La récursivité est une technique de programmation dans laquelle une fonction s'appelle elle-même.

Voici un code en Python qui calcule la factorielle d'un nombre en utilisant une boucle while VS la récursivité :

Boucle while

num: int = 5
fact: int = 1

while num > 0:
    fact = fact * num
    num = num - 1

print(fact)

Récursivité

def factorial(n: int):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

print(factorial(5))

Récursivité

  1. Écrire un programme Python qui va générer automatiquement des fichiers de log.
  2. Le premier fichier de log doit s'appeler "log.txt", le deuxième "log_1.txt", "log_2.txt"... jusqu'à log_10.txt.
  3. Essayer de gérer les générations de fichier après log_10.txt.

Pour créer un fichier, vous pouvez utiliser la fonction open() : 

import datetime
import typing

f: typing.TextIO = open("log.txt", "w")
f.write("Log file created on " + str(datetime.datetime.now()))
f.close()

Vous aurez également besoin de la librairie os pour lister les fichiers d'un répertoire, vérifier si un fichier existe et extraire l'extension du nom de fichier.
